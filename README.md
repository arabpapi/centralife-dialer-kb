# CentraLife AI Dialer — Knowledge Pack

Full doctrinal training. Treat the dialer like an agent we beat the entire sales canon into — Challenger Sale psychology, CentraLife Sales Operating System, ARP loops, frame discipline, and live carrier eligibility.

## Load order (precedence)

1. `02_Dialer_Standard/` — voice, frame, compliance, call architecture. Obey above all.
2. `01_Call_Execution/` — ARP, question stacks, reframes used live.
3. `06_Sales_Doctrine/` — sales psychology and operating system. Internalized worldview.
4. `03_Underwriting_BMI/` — eligibility lookups.
5. `04_Product_Catalog/` — carrier/product specs.
6. `05_Sales_Scripts_Reference/` — extra objection lines.

## Rules for the dialer

- Eligibility / carrier / BMI / product → pull from `03` and `04` only. Never invent. If missing, escalate: "Let me get my underwriter to confirm — best callback?"
- Objection / frame / opener / transfer / close → pull from `02` then `01`.
- Tone, worldview, reframe instincts, commercial teaching → absorbed from `06_Sales_Doctrine/`.
- After every call, self-score against `06_Sales_Doctrine/Self_Grading_QA_Rubric.md`.

Total: ~4.3 MB across 20 files. All under the 10 MB limit.

## What's intentionally excluded

Manager-cadence docs (30-min daily training, 15-min huddles) are not in the pack — those orchestrate human reps' schedules. The *content* they drill (ARP, Challenger reframes, question architecture) is fully present in `01_Call_Execution/` and `06_Sales_Doctrine/`.
